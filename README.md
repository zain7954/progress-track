# Progress Track

Progress Track is a productivity web application designed to help developers track their learning progress, projects, and goals.

It includes a dashboard, planning tools, and a progress tracking engine.

---

## Features

- User authentication
- Developer dashboard
- Planning system
- Progress tracking engine
- Clean UI with TailwindCSS

---

## Tech Stack

- React
- TypeScript
- Vite
- TailwindCSS

---

## Getting Started

Clone the repository:

git clone https://github.com/zain7954/progress-track

Go to the project folder:

cd progress-track

Install dependencies:

npm install

Run the development server:

npm run dev
